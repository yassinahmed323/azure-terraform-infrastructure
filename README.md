\# Azure Terraform Infrastructure



\## Overview

This project demonstrates Infrastructure as Code (IaC) using Terraform to provision and manage resources in Microsoft Azure.



The goal of this project is to showcase cloud infrastructure fundamentals, automation, and DevOps best practices.



\## Technologies Used

\- Terraform

\- Microsoft Azure

\- Azure Resource Manager (ARM)

\- Networking fundamentals

\- Linux



\## Architecture

This project provisions:

\- Azure Resource Group

\- Virtual Network

\- Subnet

\- (Optional) Virtual Machine



\## How to Use

1\. Authenticate to Azure

2\. Initialize Terraform

3\. Review the execution plan

4\. Apply infrastructure changes



\## What I Learned

\- Writing reusable Terraform configurations

\- Managing cloud infrastructure declaratively

\- Understanding Azure networking concepts

\- Applying DevOps workflows with Git
